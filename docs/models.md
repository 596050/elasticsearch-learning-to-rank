# Supported model formats

## Ranklib (deprecated)
Type: `model/ranklib`
All models generated by https://github.com/o19s/RankyMcRankFace

## Linear Model
Type: `model/linear`

Generates a simple linear ranker. The format is a simple `json` object:

```json
{
  "feature1" : 0.3,
  "feature2" : 0.5,
  "feature3" : 0.1
}
